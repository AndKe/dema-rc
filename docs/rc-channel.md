# RC Channel Map

|      |   |
|------|---|
| RC1  | roll |
| RC2  | pitch |
| RC3  | throttle |
| RC4  | yaw |
| RC5  | (flight mode) Left rocker to the left = RTL,<br/> mid = FBWA, right= Manual |
| RC6  | settings-button |
| RC7  | home-button |
| RC8  | takeoff/land |
| RC9  | button B |
| RC10 | button A |
| RC11 | left trigger |
| RC12 | right trigger |
| RC13 | right scrollwheel scroll left |
| RC14 | right scrollwheel scroll right |
